# 🛡️ Cyber Guardian AI

**An AI-Powered Cybersecurity Assistant for Everyone**

Cyber Guardian AI is a web-based cybersecurity tool that helps individuals and small businesses detect phishing emails, analyze suspicious URLs, learn about online threats, and improve their security awareness — all powered by AI.

---

## ✨ Features

### 📧 AI Phishing Email Analyzer
Paste an email subject and content to check if it's a phishing attempt. The AI detects urgency, suspicious phrases, fake greetings, and financial scams, then assigns a risk score (0-100).

### 🔗 URL Security Analyzer
Enter a URL to check if it's safe. The system detects IP addresses, suspicious domains, URL shorteners, missing HTTPS, and typosquatting attempts.

### 🤖 AI Security Coach
Ask any cybersecurity question and get a beginner-friendly answer. Topics include phishing, malware, ransomware, password security, social media protection, and more.

### 🧠 Security Awareness Quiz
Test your knowledge with multiple-choice questions on cybersecurity topics. Get instant feedback and explanations for each answer.

### 📊 Security Dashboard
View your analysis history, average risk scores, quiz results, and recent activity all in one place.
